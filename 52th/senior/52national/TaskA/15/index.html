解題步驟
1.安裝 phpunit
首先在專案中使用 composer 安裝 PHPUnit
composer require --dev phpunit/phpunit
2.安裝 php-xdebug
如果要能夠看到測試的 Code Coverage 就還需要安裝 php-xdebug
pecl install xdebug
3. phpunit.xml
4. 輸入 ./vendor/bin/phpunit
5. 完成
